# Fullstack Developer Portfolio

A modern, responsive single-page portfolio website for freelance work.

## Files

- `index.html` — main structure and content
- `styles.css` — layout, visual design, and responsiveness
- `script.js` — navigation interactions, active link highlighting, contact form behavior

## Run locally

Open `index.html` directly in your browser, or run a local server:

```bash
python3 -m http.server 5500
```

Then open: `http://localhost:5500`

## Quick customization

- Update your name and tagline in `index.html`
- Replace placeholder projects with your real work
- Change contact email in `index.html` (`data-recipient="akshatsheth2001@gmail.com"`)
- Adjust theme colors in `styles.css` (`:root` variables)
